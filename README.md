# Bear Images

## Description
A clean, minimalist web application designed to fetch and display random images of bears from a public image API.

## APIs Used
- [PlaceBear API](https://placebear.com) 

## Features Implemented
- Updates bear portraits without page refresh.
- Keeps images inside fixed size containers.
- Includes button with smooth hover effects.
- Features fade transitions for new photos.
- Loads high resolution images very quickly
